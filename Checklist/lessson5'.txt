With CSS Animations, we have the animation-name, the animation-duration, animation-delay, animation-iteration-count which represents the number of times
that we want the animation to run for. We can also set it to infinite to get the animation to run infinitely. Then we have the animation-fill-mode which specifies the
state that we want the animation to return to, either before, after or both

With phpmailer() we need to set the constructor when we are making use of the object to enable the use of exceptions

we need to set $MAIL->smtpAUTH to true/
