Chakra ui has a container component which has some default parameters.

Chakra UI has its own design system and every design system has its own design tokens, colors, fonts, spacing units etc. We can make use of style props to access those design tokens
that are a part of Chakra UI

For Chakra UI, we can convert spacing values into pixels by multiplying them by 4, or vice-versa by dividing them by 4.
